# Plan 1-1: Infrastruktur Next.js & UI

## Objective
Menginisialisasi framework dengan Next.js App Router, sistem pewarnaan *Dark/Light mode*, dan menyiapkan wadah *Docker Compose* untuk menampung PostgreSQL agar lingkungan dasar langsung siap jalan.

## Code Patterns (Model Resilience)
Conventions:
- File naming: Semua dalam `kebab-case` untuk dir, `kebab-case` atau `camelCase` standard untuk komponen.
