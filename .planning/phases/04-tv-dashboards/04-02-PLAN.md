# Plan 4-2: TV Layout and Theme Support

## Objective
Set up the base layout for public TV displays, including support for URL-based theme switching (Dark/Light).

## Code Patterns
- Use a root `layout.tsx` under `src/app/tv` to handle global TV styles.
- Extract `theme` from URL query parameters (e.g., `?theme=dark`).
