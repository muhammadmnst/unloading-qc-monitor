# Plan 3-3: Material Data Grids and Auto-polling

## Objective
Implement the actual data display tables for each specific material. The tables will auto-fetch every 10 seconds, hide completed logs by default (with a toggle), and have dedicated action buttons on each row.

## Code Patterns
Conventions:
- Client Components: Require `"use client"` along with `useEffect` with `setInterval` to periodically invoke the Server Action `getDashboardLogs`.
